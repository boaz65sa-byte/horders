## 🎯 שימושים מהירים

### 📝 להזמין מספק (60 שניות)

1. ✅ **בחר ספק** מהרשימה
2. ✅ **בחר תאריך אספקה** (ברירת מחדל: מחר)
3. ✅ **הכנס כמויות** ליד המוצרים
4. ✅ **שנה יחידה** אם צריך (ק״ג/יחידה/ארגז)
5. ✅ **בחר אופן שליחה**: WhatsApp / Email / שניהם
6. ✅ **סמן/בטל** "הצג מחירים"
7. ✅ **לחץ "שלח הזמנה"**
8. ✅ **WhatsApp/Email נפתחים אוטומטית**
9. ✅ **שלח!**

---

### ➕ להוסיף ספק חדש (30 שניות)

```
טאב "👥 ספקים"
→ מלא: שם, טלפון, אימייל, קטגוריה
→ לחץ "➕ הוסף ספק"
→ מוכן! ✅
```

---

### ✏️ לערוך פרטי ספק (30 שניות)

```
טאב "👥 ספקים"
→ בחר ספק → לחץ "✏️ ערוך"
→ שנה פרטים
→ לחץ "💾 שמור שינויים"
→ עודכן! ✅
```

---

### 📦 להוסיף מוצר (30 שניות)

```
טאב "📦 מוצרים"
→ בחר ספק
→ הכנס: שם, מחיר, יחידה
→ לחץ "➕ הוסף מוצר"
→ נוסף! ✅
```

---

### 📂 לייבא מוצרים מExcel (2 דקות)

1. ✅ **הכן קובץ Excel/CSV** בפורמט:
   ```
   שם מוצר | מחיר | יחידה
   חסה     | 8    | ק"ג
   עגבניות  | 12   | ק"ג
   ```
2. ✅ **טאב "📦 מוצרים"**
3. ✅ **בחר ספק לייבוא**
4. ✅ **לחץ "📄 בחר קובץ Excel/CSV"**
5. ✅ **העלה קובץ**
6. ✅ **בדוק תצוגה מקדימה**
7. ✅ **לחץ "⬆️ ייבא מוצרים"**
8. ✅ **כל המוצרים יובאו!**

## 🆕 תכונות חדשות בגרסה 3.0

### 1️⃣ ייבוא מExcel
**למה זה מעולה?**
- ייבוא עשרות מוצרים בבת אחת
- עדכון מחירים מהיר
- חיסכון בזמן עצום

### 2️⃣ מחירים כן/לא
**איך זה עוזר?**
- הזמנות עם מחירים → לניהול עלויות
- הזמנות בלי מחירים → שליחה לספקים
- גמישות מקסימלית

### 3️⃣ WhatsApp או Email או שניהם
**מה זה נותן?**
- WhatsApp → מהיר ונוח
- Email → תיעוד רשמי
- שניהם → הכי מקיף
