## ✨ תכונות עיקריות

### 🎯 תכונות מתקדמות בגרסה 3.0!

#### 1️⃣ **ייבוא מוצרים מקבצי Excel/CSV** 📂
- העלאה מהירה של מחירונים שלמים מספקים
- תמיכה בקבצי Excel (.xlsx, .xls) וCSV
- תצוגה מקדימה של המוצרים לפני הייבוא
- פורמט פשוט: שם מוצר | מחיר | יחידה
- ייבוא עשרות מוצרים בלחיצה אחת!

**איך להשתמש:**
1. בטאב "מוצרים" - בחר ספק
2. לחץ "בחר קובץ Excel/CSV"
3. העלה את הקובץ
4. בדוק את התצוגה המקדימה
5. לחץ "ייבא מוצרים"

**פורמט קובץ Excel דוגמה:**
```
שם מוצר    | מחיר  | יחידה
חסה        | 8     | ק"ג
עגבניות     | 12    | ק"ג
מלפפונים    | 10    | ק"ג
ביצים L    | 18    | יחידה
```

#### 2️⃣ **בחירה: הזמנה עם או בלי מחירים** 💰
- צ'קבוקס נוח בעמוד ההזמנה
- בחר אם להציג מחירים בהודעת WhatsApp/Email
- שמירה אוטומטית של ההעדפה
- שימושי למצבים שונים (לקוחות/ספקים)

**דוגמאות:**

*עם מחירים:*
```
• חסה: 3 ק״ג = ₪24.00
• עגבניות: 5 יחידה = ₪60.00
━━━━━━━━━━━━━━━
סה״כ: ₪84.00
```

*בלי מחירים:*
```
• חסה: 3 ק״ג
• עגבניות: 5 יחידה
```

#### 3️⃣ **בחירת אופן שליחה: WhatsApp / Email / שניהם** 📱📧
- בחר איך לשלוח את ההזמנה
- אופציות: WhatsApp, Email, או שניהם
- WhatsApp - פתיחה ישירה עם הודעה מעוצבת
- Email - פתיחת mailto עם נושא וגוף מוכן
- "שניהם" - שליחת ההזמנה בשני הערוצים

## 📱 איך להתחיל?

### אופציה 1️⃣ - פתיחה במחשב (מהיר!)
1. הורד את הקבצים
2. לחיצה כפולה על **index.html**
3. המערכת תיפתח בדפדפן - **מוכן!** ✅

### אופציה 2️⃣ - פתיחה באייפון/אנדרואיד (מומלץ!)
1. העלה את 4 הקבצים ל: **https://app.netlify.com/drop**
   - index.html
   - styles.css
   - app.js
   - manifest.json
2. קבל קישור אוטומטית
3. פתח את הקישור בטלפון
4. Safari/Chrome → Share → **Add to Home Screen**
5. עכשיו יש לך אפליקציה במסך הבית! 🎉

### אופציה 3️⃣ - שליחה במייל
1. שלח את index.html למייל שלך
2. פתח במייל בטלפון
3. לחץ על הקובץ → "פתח ב-Safari"

## 🎯 מדריך שימוש

### 📝 יצירת הזמנה (60 שניות)

1. **פתח את המערכת**
2. **טאב "🛍️ הזמנה"**
3. **בחר ספק** מהרשימה הנפתחת
4. **בחר תאריך אספקה** (ברירת מחדל: מחר)
5. **הכנס כמויות** ליד המוצרים הרצויים
6. **שנה יחידה** אם צריך (ק״ג/יחידה/ארגז)
7. **בחר אופן שליחה**: WhatsApp / Email / שניהם
8. **סמן/בטל סימון** "הצג מחירים" לפי הצורך
9. **לחץ "שלח הזמנה"**
10. **WhatsApp/Email ייפתחו אוטומטית** עם ההודעה המעוצבת
11. **שלח!** ✅

### ➕ הוספת ספק חדש

1. **טאב "👥 ספקים"**
2. **מלא פרטים:** שם, טלפון, אימייל, קטגוריה
3. **לחץ "➕ הוסף ספק"**
4. **הספק נוסף!** ✅

### ✏️ עריכת פרטי ספק

1. **טאב "👥 ספקים"**
2. **בחר ספק** ולחץ **"✏️ ערוך"**
3. **שנה פרטים** (שם, טלפון, אימייל, קטגוריה)
4. **לחץ "💾 שמור שינויים"**

### 📦 הוספת מוצר ידנית

1. **טאב "📦 מוצרים"**

3. **בחר ספק**
4. **הכנס:** שם מוצר, מחיר, יחידה
5. **לחץ "➕ הוסף מוצר"**
6. **המוצר נוסף!** ✅

### 📂 ייבוא מוצרים מExcel

1. **טאב "📦 מוצרים"**
