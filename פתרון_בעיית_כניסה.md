## ✅ פתרונות מהירים

### פתרון 1️⃣ - בדיקה ראשונית (מומלץ!)

1. **פתח את הקובץ:** `בדיקת_כניסה.html`
2. **לחץ על הכפתורים:**
   - "בדוק localStorage" - וודא שעובד
   - "בדוק קודים" - ראה מה שמור
   - "נקה הכל" - אם יש בעיה

3. **חזור ל-index.html** ונסה שוב

---

### פתרון 2️⃣ - ניקוי נתונים

**דרך A - דרך הקובץ בדיקת_כניסה.html:**
1. פתח `בדיקת_כניסה.html`
2. לחץ "נקה הכל"
3. חזור ל-`index.html`
4. נסה להיכנס עם `123456`

**דרך B - דרך Console:**
1. פתח את `index.html`
2. לחץ **F12** (Developer Tools)
3. לך ל-**Console**
4. הקלד:
   ```javascript
   localStorage.clear();
   location.reload();
   ```
5. נסה להיכנס עם `123456`

## 🔍 בדיקות מתקדמות

### בדיקה 1 - האם הכפתור עובד?

1. פתח `index.html`
2. לחץ **F12** → Console
3. הקלד:
   ```javascript
   document.getElementById('login-btn').onclick = function() {
       alert('כפתור עובד!');
   };
   ```
4. לחץ על כפתור הכניסה
5. אם מופיע alert - הכפתור עובד

### בדיקה 2 - האם הקוד מתקבל?

1. פתח `index.html`
2. לחץ **F12** → Console
3. הקלד:
   ```javascript
   document.getElementById('login-code').addEventListener('input', function(e) {
       console.log('קוד:', e.target.value);
   });
   ```
4. הקלד משהו בשדה הקוד
5. ראה ב-Console אם מודפס

### בדיקה 3 - מה הקוד השמור?

1. פתח `index.html`
2. לחץ **F12** → Console
3. הקלד:
   ```javascript
   console.log('Admin code:', localStorage.getItem('adminCode') || '123456 (default)');
   console.log('Employee code:', localStorage.getItem('employeeCode') || '654321 (default)');
   ```
4. ראה מה מודפס

## 🛠️ פתרון ידני - עקוף את מסך הכניסה

אם שום דבר לא עוזר, אפשר לעקוף את מסך הכניסה זמנית:

1. פתח `index.html`
2. לחץ **F12** → Console
3. הקלד:
   ```javascript
   localStorage.setItem('currentUser', JSON.stringify({role: 'admin', code: '123456'}));
   location.reload();
   ```
4. המערכת תיפתח כמנהל

**⚠️ זה רק זמני! אחרי שנכנסת, לך להגדרות ושנה את הקודים**

### 🔴 בעיה: "קוד שגוי! נסה שוב"

   - `123456` (מנהל)
   - `654321` (עובד)

   ```javascript

   ```

### 🔴 בעיה: הכפתור לא מגיב

**פתרון:**
1. נסה ללחוץ **Enter** במקום הכפתור
2. אם לא עובד, רענן את הדף (F5)
3. אם עדיין לא, נקה Cache:
   - Chrome: Ctrl+Shift+Delete
   - בחר "Cached images and files"
   - לחץ "Clear data"

**פתרון:**

