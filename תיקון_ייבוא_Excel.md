## 📋 איך להשתמש (מעודכן)

### שלב 1: הכן קובץ Excel/CSV
הפורמט הנכון:

```
שם מוצר  | מחיר | יחידה
---------|------|-------
חסה      | 8    | ק״ג
עגבניות   | 12   | ק״ג
מלפפונים  | 10   | ק״ג
```

**חשוב:**
- עמודה A: שם המוצר (חובה, טקסט)
- עמודה B: מחיר (חובה, מספר)
- עמודה C: יחידה (אופציונלי, ברירת מחדל: ק״ג)

**ניתן לכלול שורת כותרת או להתחיל ישירות מהנתונים!**

### שלב 2: טען את הקובץ
1. טאב "📦 מוצרים"
2. בחר ספק מהרשימה
3. לחץ "📄 בחר קובץ Excel/CSV"
4. בחר את הקובץ שלך

### שלב 3: בדוק תצוגה מקדימה
- המערכת תציג טבלה עם כל המוצרים
- וודא שהשמות, מחירים ויחידות נכונים

### שלב 4: ייבא
- לחץ "⬆️ ייבא מוצרים"
- תקבל הודעה: "✅ יובאו X מוצרים בהצלחה!"
- המערכת תנקה הכל אוטומטית

   ✅ מנקה את הקודם

## 🆘 פתרון בעיות

### ❌ "הקובץ ריק או לא מכיל נתונים"
**סיבות אפשריות:**
- הקובץ באמת ריק
- כל השורות ריקות
- הפורמט לא נכון

**פתרון:**
1. פתח את הקובץ ב-Excel
2. ודא שיש נתונים בעמודות A, B (ואופציונלי C)
3. שמור מחדש
4. נסה שוב

### ❌ "לא נמצאו מוצרים תקינים בקובץ"
**סיבות אפשריות:**
- עמודה B לא מכילה מספרים
- השמות בעמודה A ריקים
- הפורמט לא נכון

**פתרון:**
1. בדוק שעמודה A מכילה שמות
2. בדוק שעמודה B מכילה מספרים בלבד
3. השתמש ב-`דוגמת_Excel_נכונה.csv` כדוגמה

### ❌ המוצרים מהקובץ הקודם נשארו
**זה לא אמור לקרות יותר!**

אבל אם זה קורה:
1. רענן את הדף (F5)
2. נסה שוב
3. אם זה ממשיך, דווח על הבעיה

## 🔐 קוד מקור - clearExcelPreview()

```javascript
clearExcelPreview() {
    // Clear preview display
    const preview = document.getElementById('excel-preview');
    if (preview) {
        preview.style.display = 'none';
    }
    
    const content = document.getElementById('preview-content');
    if (content) {
        content.innerHTML = '';
    }
    
    // Clear file input
    const fileInput = document.getElementById('excel-file');
    if (fileInput) {
        fileInput.value = '';
    }
    
    // Clear file name display
    const fileName = document.getElementById('file-name');
    if (fileName) {
        fileName.textContent = '';
    }
    
    // Clear supplier selection
    const supplierSelect = document.getElementById('excel-supplier-select');
    if (supplierSelect) {
        supplierSelect.value = '';
    }
    
    // Clear data
    this.excelPreviewData = null;
}
```
